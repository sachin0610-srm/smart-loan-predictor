# Loan Eligibility Genius

## Overview

Loan Eligibility Genius is a smart web application designed to help users check their loan eligibility quickly and efficiently.

The system analyzes user financial details such as:

* Income
* Credit Score
* Employment Status
* Existing Loans
* Repayment Capacity
* Loan Type Preference

and provides an eligibility prediction along with useful financial insights.

---

## Features

* Loan eligibility prediction
* User-friendly dashboard
* Real-time eligibility checking
* Secure authentication
* Database integration using Supabase
* Responsive modern UI
* Admin and user management

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/loan-eligibility-genius.git
cd loan-eligibility-genius
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
